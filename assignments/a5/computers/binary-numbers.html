Binary numbers

Definition: Binary numbers are numbers represented with a binary system. Binary is a numeric system that only uses two digits; 0 and 1. Humans are used to decimal number system which has ten unique digits from 0 to 9. Computers operate in binary, meaning they store data and perform calculations using only zeros and ones. A single binary digit can only represent True (1) or False (0) in boolean logic. However, multiple binary digits can be used to represent large numbers and perform complex functions.
